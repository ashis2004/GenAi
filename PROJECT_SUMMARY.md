## 📋 Setup and Usage

### Quick Start (3 Steps)

```bash
# 1. Clone and setup
git clone https://github.com/ashis2004/GenAi.git
cd GenAi
python setup.py

# 2. Activate environment
source genai_env/bin/activate  # Linux/Mac
# genai_env\Scripts\activate   # Windows

# 3. Start exploring
jupyter notebook finetuning/mistral-lora.ipynb
```
